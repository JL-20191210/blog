查看系统配置

在 Linux 系统上查看设备信息通常可以通过多种工具和命令实现。以下是对常用方法的总结：

1. 查看 CPU 信息

  • 命令：

    lscpu

    输出内容：

    • CPU 架构（如 x86_64 或 arm64）。
    • 核心数量、线程数、频率等信息。

  • 查看详细 CPU 信息：

    cat /proc/cpuinfo

    包含每个处理器核心的详细数据。

2. 查看内存信息

  • 命令：

    free -h

    输出内容：

    • 总内存、已用内存、可用内存，单位为人类可读格式（如 MB/GB）。

  • 更详细的内存信息：

    cat /proc/meminfo

    提供内存分页、缓冲区等详细内容。

3. 查看磁盘信息

  • 查看分区和挂载信息：

    df -h

    输出内容：

    • 每个分区的大小、已用空间、可用空间以及挂载点。

  • 查看硬盘设备信息：

    lsblk

    显示块设备的层次结构。

  • 查看具体设备信息（如型号、容量）：

    sudo fdisk -l

    或

    sudo parted -l

4. 查看网络信息

  • 显示网络接口信息：

    ip addr show

    或

    ifconfig

  • 查看网络硬件信息：

    lspci | grep -i ethernet

5. 查看系统版本和发行版信息

  • 命令：

    lsb_release -a

    或查看 /etc/os-release 文件：

    cat /etc/os-release

  • 查看内核版本：

    uname -r

6. 查看设备架构

  • 命令：
    uname -m
    输出设备架构，如 x86_64、armv7l、aarch64。

7. 查看 GPU 信息

  • 列出显卡信息：

    lspci | grep -i vga

  • 使用专用工具查看显卡细节：

    • NVIDIA 卡：nvidia-smi
    • AMD 卡：glxinfo | grep OpenGL

8. 查看所有硬件信息

  • 命令：

    lshw

    或获取特定类型信息：

    lshw -class <类型>

    类型如 memory、cpu、disk 等。

  • 快速列出所有 PCI 设备：

    lspci

  • 列出所有 USB 设备：

    lsusb

总结表格

信息类型命令
CPU 信息lscpu, cat /proc/cpuinfo
内存信息free -h, cat /proc/meminfo
磁盘信息df -h, lsblk, fdisk -l
网络信息ip addr show, ifconfig
系统版本lsb_release -a, uname -r
设备架构uname -m
GPU 信息`lspci
全部硬件lshw, lspci, lsusb

通过以上命令，可以快速全面地了解 Linux 系统上的硬件和系统配置。
