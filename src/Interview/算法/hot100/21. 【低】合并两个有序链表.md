# [21. 合并两个有序链表](https://leetcode.cn/problems/merge-two-sorted-lists/)

## 题目描述

> 将两个升序链表合并为一个新的 **升序** 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 
>
>  
>
> **示例 1：**
>
> ![img](assets/merge_ex1.jpg)
>
> ```
> 输入：l1 = [1,2,4], l2 = [1,3,4]
> 输出：[1,1,2,3,4,4]
> ```
>
> **示例 2：**
>
> ```
> 输入：l1 = [], l2 = []
> 输出：[]
> ```
>
> **示例 3：**
>
> ```
> 输入：l1 = [], l2 = [0]
> 输出：[0]
> ```

## 代码示例

```java
class Solution {
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        // 如果 list1 为空，直接返回 list2
        if (list1 == null) {
            return list2;
        }
        // 如果 list2 为空，直接返回 list1
        else if (list2 == null) {
            return list1;
        }
        // 如果 list1 当前节点的值小于 list2 当前节点的值
        else if (list1.val < list2.val) {
            // 递归地将 list1.next 和 list2 合并，设置 list1.next
            list1.next = mergeTwoLists(list1.next, list2);
            // 返回当前 list1（因为 list1 的节点值较小，所以 list1 的节点排在前面）
            return list1;
        }
        // 如果 list2 当前节点的值小于等于 list1 当前节点的值
        else {
            // 递归地将 list1 和 list2.next 合并，设置 list2.next
            list2.next = mergeTwoLists(list1, list2.next);
            // 返回当前 list2（因为 list2 的节点值较小或等于 list1，所以 list2 的节点排在前面）
            return list2;
        }
    }
}

```
