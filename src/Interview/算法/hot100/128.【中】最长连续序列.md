# [128. 最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/)

## 题目描述

> 给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。
>
> 请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。
>
>  
>
> **示例 1：**
>
> ```
> 输入：nums = [100,4,200,1,3,2]
> 输出：4
> 解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。
> ```
>
> **示例 2：**
>
> ```
> 输入：nums = [0,3,7,2,5,8,4,6,0,1]
> 输出：9
> ```
>
> **示例 3：**
>
> ```
> 输入：nums = [1,0,1,2]
> 输出：3
> ```

## 示例代码
```java
class Solution {
    public int longestConsecutive(int[] nums) {
        // 用一个 HashSet 存放所有数字，便于 O(1) 时间判断某个数是否存在
        Set<Integer> nums_set = new HashSet<>();

        // 将数组中所有数字放入集合（去重）
        for (int num : nums) {
            nums_set.add(num);
        }

        // 用来记录最长连续序列的长度
        int stepSum = 0;

        // 遍历每个数字，尝试以它作为“连续序列的起点”
        for (int num : nums_set) {

            // 如果 num-1 不存在，说明 num 是某个连续序列的“起点”
            if (!nums_set.contains(num - 1)) {

                int cur = num;   // 当前正在统计的数字
                int step = 1;    // 当前连续序列的长度（至少包含 num 自己）

                // 一直往右扩展，找 cur+1、cur+2、cur+3 ...
                while (nums_set.contains(cur + 1)) {
                    cur += 1;
                    step += 1;   // 每找到一个连续的数，长度+1
                }

                // 更新最长连续长度
                stepSum = Math.max(stepSum, step);
            }
        }

        // 返回最长连续序列的长度
        return stepSum;
    }
}
```
