# [42. 接雨水](https://leetcode.cn/problems/trapping-rain-water/)

## 题目描述

> 给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。
>
>  
>
> **示例 1：**
>
> ![img](assets/rainwatertrap.png)
>
> ```
> 输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
> 输出：6
> 解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 
> ```
>
> **示例 2：**
>
> ```
> 输入：height = [4,2,0,3,2,5]
> 输出：9
> ```

## 代码示例

> 其实就是算了一下高度差

```java

class Solution {
    public int trap(int[] height) {
        int n = height.length;
        
        // 如果高度数组为空，则没有水可以接，返回 0
        if(n == 0) return 0;

        // 初始化 leftMax 数组，记录每个位置左边的最大高度
        int[] leftMax = new int[n];
        leftMax[0] = height[0]; // 左侧第一个位置的最大高度就是它本身

        // 填充 leftMax 数组，leftMax[i] 为从左到右的最大高度
        for (int i = 1; i < n; i++) {
            leftMax[i] = Math.max(leftMax[i-1], height[i]);
        }

        // 初始化 rightMax 数组，记录每个位置右边的最大高度
        int[] rightMax = new int[n];
        rightMax[n-1] = height[n-1]; // 右侧最后一个位置的最大高度就是它本身

        // 填充 rightMax 数组，rightMax[i] 为从右到左的最大高度
        for (int i = n - 2; i >= 0; i--) {
            rightMax[i] = Math.max(rightMax[i+1], height[i]);
        }

        // 计算总的接水量
        int ans = 0;
        for (int i = 0; i < n; i++) {
            // 当前位置的水量是该位置的左侧和右侧最大值的较小值减去当前高度
            ans += Math.min(leftMax[i], rightMax[i]) - height[i];
        }

        // 返回计算出来的总接水量
        return ans;
    }
}

```
