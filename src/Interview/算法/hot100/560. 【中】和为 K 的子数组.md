# [560. 和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/)

## 题目描述

> 给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回 *该数组中和为 `k` 的子数组的个数* 。
>
> 子数组是数组中元素的连续非空序列。
>
>  
>
> **示例 1：**
>
> ```
> 输入：nums = [1,1,1], k = 2
> 输出：2
> ```
>
> **示例 2：**
>
> ```
> 输入：nums = [1,2,3], k = 3
> 输出：2
> ```

## 代码示例

```java
import java.util.HashMap;

class Solution {
    public int subarraySum(int[] nums, int k) {
        // 创建一个 HashMap，用来记录每个前缀和出现的次数
        HashMap<Integer, Integer> prefixSumCount = new HashMap<>();

        // 初始化，前缀和为0出现一次，处理一些特殊情况（比如子数组从开头开始）
        prefixSumCount.put(0, 1);

        // 当前的前缀和
        int currentSum = 0;

        // 记录和为 k 的子数组个数
        int count = 0;

        // 遍历数组 nums
        for (int num : nums) {
            // 更新当前的前缀和
            currentSum += num;

            // 检查当前前缀和减去 k 是否在 prefixSumCount 中出现过
            // 如果出现过，说明存在一个子数组的和为 k
            if (prefixSumCount.containsKey(currentSum - k)) {
                // 将符合条件的子数组个数加入到 count 中
                count += prefixSumCount.get(currentSum - k);
            }

            // 将当前的前缀和加入到 HashMap 中
            // 如果当前的前缀和已经存在，则加1；如果不存在，则初始化为1
            prefixSumCount.put(currentSum, prefixSumCount.getOrDefault(currentSum, 0) + 1);
        }

        // 返回最终的结果：和为 k 的子数组个数
        return count;
    }
}

```
