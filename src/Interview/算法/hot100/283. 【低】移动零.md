# [283. 移动零](https://leetcode.cn/problems/move-zeroes/)

## 题目描述

> 给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。
>
> **请注意** ，必须在不复制数组的情况下原地对数组进行操作。
>
>  
>
> **示例 1:**
>
> ```
> 输入: nums = [0,1,0,3,12]
> 输出: [1,3,12,0,0]
> ```
>
> **示例 2:**
>
> ```
> 输入: nums = [0]
> 输出: [0]
> ```

## 代码示例

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int n = nums.length;
        int left = 0, right = 0;

        // 遍历数组，right 用来遍历每个元素，left 用来记录下一个非零元素的位置
        while (right < n) {
            // 如果当前元素不为零，则交换元素并把 left 指针向右移动
            if (nums[right] != 0) {
                swap(nums, left, right);
                left++;  // 将 left 移动到下一个位置
            }
            right++;  // right 指针始终向右移动
        }
    }

    // 用于交换数组 nums 中两个索引的值
    public void swap(int[] nums, int left, int right) {
        int temp = nums[left];
        nums[left] = nums[right];
        nums[right] = temp;
    }
}
```
