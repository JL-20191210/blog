# [49. 字母异位词分组](https://leetcode.cn/problems/group-anagrams/)

## 题目描述

> 给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。
>
>  
>
> **示例 1:**
>
> **输入:** strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
>
> **输出:** [["bat"],["nat","tan"],["ate","eat","tea"]]
>
> **解释：**
>
> - 在 strs 中没有字符串可以通过重新排列来形成 `"bat"`。
> - 字符串 `"nat"` 和 `"tan"` 是字母异位词，因为它们可以重新排列以形成彼此。
> - 字符串 `"ate"` ，`"eat"` 和 `"tea"` 是字母异位词，因为它们可以重新排列以形成彼此。
>
> **示例 2:**
>
> **输入:** strs = [""]
>
> **输出:** [[""]]
>
> **示例 3:**
>
> **输入:** strs = ["a"]
>
> **输出:** [["a"]]

## 示例代码

```java
class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        // 创建一个 HashMap，用于存储以字母排序后的字符串作为 key，对应的所有字母异位词作为 value
        Map<String, List<String>> map = new HashMap<String, List<String>>();
        
        // 遍历输入的字符串数组 strs
        for (String str : strs) {
            // 将当前字符串转换为字符数组
            char[] array = str.toCharArray();
            
            // 对字符数组进行排序，这样相同字母组成的字符串会变成相同的顺序
            Arrays.sort(array);
            
            // 将排序后的字符数组转回字符串，作为字母异位词的 key
            String key = new String(array);
            
            // 使用 map.getOrDefault 获取当前 key 对应的字母异位词列表，如果没有则返回一个新的 ArrayList
            List<String> list = map.getOrDefault(key, new ArrayList<String>());
            
            // 将当前字符串添加到对应的字母异位词列表中
            list.add(str);
            
            // 更新 map 中当前 key 对应的字母异位词列表
            map.put(key, list);
        }

        // 返回 map 中所有字母异位词列表的值
        return new ArrayList<List<String>>(map.values());
    }
}

```
