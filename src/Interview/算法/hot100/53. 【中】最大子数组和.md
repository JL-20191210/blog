# [53. 最大子数组和](https://leetcode.cn/problems/maximum-subarray/)

## 题目描述

> 给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。
>
> **子数组**是数组中的一个连续部分。
>
>  
>
> **示例 1：**
>
> ```
> 输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
> 输出：6
> 解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。
> ```
>
> **示例 2：**
>
> ```
> 输入：nums = [1]
> 输出：1
> ```
>
> **示例 3：**
>
> ```
> 输入：nums = [5,4,-1,7,8]
> 输出：23
> ```

## 代码描述

```java
class Solution {
    public int maxSubArray(int[] nums) {
        // pre 记录当前子数组的和，maxAns 记录最大子数组和
        int pre = 0, maxAns = nums[0];

        // 遍历数组中的每一个元素
        for (int x : nums) {
            // pre = Math.max(pre + x, x) 选择是否将当前元素加入现有子数组（如果加入后和更大），或者重新从当前元素开始
            pre = Math.max(pre + x, x);
            // maxAns 记录到目前为止的最大子数组和
            maxAns = Math.max(maxAns, pre);
        }

        return maxAns;
    }
}
```

> 如果pre加上x后小于x，那么说明pre起副作用
