# [15. 三数之和](https://leetcode.cn/problems/3sum/)

## 题目描述

> 给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。
>
> **注意：**答案中不可以包含重复的三元组。
>
>  
>
>  
>
> **示例 1：**
>
> ```
> 输入：nums = [-1,0,1,2,-1,-4]
> 输出：[[-1,-1,2],[-1,0,1]]
> 解释：
> nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
> nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
> nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
> 不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
> 注意，输出的顺序和三元组的顺序并不重要。
> ```
>
> **示例 2：**
>
> ```
> 输入：nums = [0,1,1]
> 输出：[]
> 解释：唯一可能的三元组和不为 0 。
> ```
>
> **示例 3：**
>
> ```
> 输入：nums = [0,0,0]
> 输出：[[0,0,0]]
> 解释：唯一可能的三元组和为 0 。
> ```

## 示例代码

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        int n = nums.length;
        
        // 先对数组进行排序，便于后续使用双指针
        Arrays.sort(nums);
        
        // 用于存储结果的列表
        List<List<Integer>> ans = new ArrayList<List<Integer>>();

        // 遍历数组中的每一个元素，作为三元组的第一个元素
        for (int i1 = 0; i1 < n; i1++) {
            // 如果当前元素与前一个元素相同，跳过，避免重复三元组
            if (i1 > 0 && nums[i1] == nums[i1 - 1]) continue;

            int i3 = n - 1; // 设定双指针，i3指向数组末尾
            int target = -nums[i1]; // 计算目标值，即需要找到的 nums[i2] + nums[i3] 的和

            // 第二个指针 i2 从 i1 + 1 开始向右移动
            for (int i2 = i1 + 1; i2 < n; i2++) {
                // 如果当前元素与前一个元素相同，跳过，避免重复三元组
                if (i2 > i1 + 1 && nums[i2] == nums[i2 - 1]) continue;

                // 使用双指针方法，调整 i3 位置，保证 nums[i2] + nums[i3] <= target
                while (i2 < i3 && nums[i2] + nums[i3] > target) {
                    i3--;
                }

                // 如果 i2 与 i3 重合，说明已经找不到符合条件的三元组，跳出循环
                if (i2 == i3) {
                    break;
                }

                // 如果找到一组三元组满足条件，则加入结果列表
                if (nums[i2] + nums[i3] == target) {
                    List<Integer> list = new ArrayList<>();
                    list.add(nums[i1]);
                    list.add(nums[i2]);
                    list.add(nums[i3]);
                    ans.add(list);
                }
            }
        }

        // 返回结果列表
        return ans;
    }
}
```
