# 反射总结

## 1.反射的作用

&#9312;获取任意一个类中的所有信息

&#9313;结合配置文件动态创建对象

## 2.获得class字节码文件对象的三种方式

&#9312;Class.forName("全类名")

## 3.如何获取构造方法、成员方法、成员变量

- get:获取
- set:设置
- Constructor:构造方法
- Parameter:参数
- Field:成员变量
- Modifiers:修饰符
- Method:方法
- Declared:私有的
